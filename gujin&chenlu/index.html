05 / 家庭版一权益

2个保证入住权 + 4个优先入住权

家庭版一的重点，是把未来养老社区选择从个人扩展到家庭。后续是否真正入住，可在符合年龄、健康评估和社区规则时再做决定。

家庭版一：面向家庭的入住权益

当前300万方案可按500万家庭版一权益对接：提前保留家庭成员未来入住泰康之家养老社区的资格和优先顺位。

权益主体更像“家庭选择权”

家庭版一不是只给一个人留名额，而是围绕家庭成员来配置入住权益。

  • 2个终身保证入住权，可在确认函约定范围内指定权益人。
  • 可选范围通常包括投保人、被保险人、配偶、父母、子女等家庭成员。
  • 4个优先入住权，重点为父母辈未来养老选择预留优先顺位。
  • 最终权益主体、指定方式和变更规则，以确认函文本为准。

保证入住权的价值

保证入住权解决的是未来真正想住时的确定性和排队优先级。

  • 通过正式入住申请及社区评估后，按确认函约定享有入住安排。
  • 在排队顺位上，保证权行权客户通常优先于优先权客户及普通客户。
  • 资料口径中，保证入住权在通过评估后按约定期限安排入住。
  • 相关文件签署、费用缴纳和社区管理规定仍需同步满足。

全国、终身、CCRC全业态

家庭版一的核心不是某一个房间，而是未来在泰康之家体系内保留选择权。

  • 全国范围内泰康之家社区可选，具体以确认函和当期规则为准。
  • 终身保证入住权，避免未来年龄越来越大时选择权变窄。
  • CCRC全业态覆盖独立生活、协助生活、专业护理、康复照护等连续服务。
  • 入住前仍需符合年龄、健康评估、房源和社区管理要求。
06 / 医养康宁权益

不止养老社区，也是一套健康与生活服务入口

这笔家庭底仓除了未来现金流，也会把家庭带入泰康的医、养、康、宁服务体系。具体可用权益以客户资格和当期服务规则为准。

湖北医养康宁生态

围绕武汉和湖北本地，形成养老、康复、医疗、口腔、田园生活和生命关怀的一体化资源入口。

  • 楚园养老社区：未来养老社区入住与长期生活场景。
  • 楚园康复医院：康复、护理、安宁疗护等医疗照护资源。
  • 泰康同济（武汉）医院：本地综合医疗资源连接。
  • 泰康口腔、京山农场、仙鹤湖：覆盖口腔健康、休闲旅居、生命关怀等场景。

绿通与医疗协助

在符合服务资格和病种条件时，围绕诊前、诊中、诊后提供就医流程协助。

  • 诊前：精确分诊、就医咨询、专家推荐、异地就医方向判断。
  • 诊中：专家门诊、专家二诊、医学陪诊、检查加急、专家病房等协助。
  • 诊后：专家康复、住院护工、出院陪护、远程康护、上门照护等支持。
  • 服务目标是帮助家庭更快找到方向、少跑流程、减少关键时刻的信息差。

在线多学科会诊

围绕指定重大疾病，可协调多学科专家线上会诊，帮助家庭获得更完整的诊疗意见。

  • 安排不少于3名相关学科专家进行在线视频会诊。
  • 专家讨论后形成MDT会诊意见，供客户及主治医生参考。
  • 资料显示服务资源覆盖1000+三级医院、5000+专家资源和13大科室。
  • 适合肿瘤、复杂重疾、诊疗方案需要多学科判断的场景。

住院垫付

指定医疗险有效保单被保险人，在符合等待期、病种、医院和审核条件后，可申请住院垫付服务。

  • 适用场景包括资料所列40种轻症、20种中症、120种重疾。
  • 入住泰康绿通医院并通过资格及报案审核后，可申请垫付。
  • 资料显示最多3次住院垫付，每次最高20万元。
  • 建议至少在住院前一周发起申请，以免审核进度影响使用。

楚惠卡本地权益

楚惠卡按普卡、金卡、钻石卡分层，适合把高客权益落到湖北本地可用服务上。

  • 专科项目：皮肤、中医、康复、眼科、核医学、疼痛、儿保、全科等。
  • 口腔权益：洁牙、检查、牙科体检报告、口腔CT/全景片等分层权益。
  • 健康管理：营养代谢、人体成分分析、入住式体检房费折扣等。
  • 生活权益：京山农场住宿餐饮、活动课程、仙鹤湖生命关怀服务等。

私人医生与家庭健康管理

私人医生权益更适合日常长期使用，把体检、报告、健康追踪和就医方向管理起来。

  • 建立健康档案，沉淀本人及家庭成员的健康信息。
  • 体检报告深度解读，帮助看懂指标和后续检查方向。
  • 按等级匹配权益体检、上门问诊、每月健康追踪等服务。
  • 必要时可进一步衔接泰康医疗、绿通、康复等资源。
07 / 幸福有约六大私享

不只是权益名目，而是长期家庭服务菜单

六大私享可理解为泰康为高客家庭提供的延展服务体系，覆盖健康、财富、教育、艺术、慈善和礼遇。具体可用项目、次数和资格，以客户等级及当期权益规则为准。

健康私享

把家庭健康管理前置，而不是等生病后才临时找资源。

  • 家庭医生：健康档案、体检报告解读、健康咨询和长期追踪。
  • 体检服务：按客户等级匹配权益体检或体检相关服务。
  • 泰医通：在符合规则时连接就医协助、医疗资源和健康服务入口。
  • 海外服务：部分权益可延展至海外医疗咨询、国际二诊等方向，按资格和服务手册执行。

财富私享

适合企业主家庭把保险、资产、法律和传承问题放在同一张图里看。

  • 财富顾问：围绕保险、基金、信托、资管、公益慈善等做资源连接。
  • 投资沙龙：了解宏观、市场、资产配置和长期财富管理观点。
  • 法商沙龙与线上法律顾问：关注婚姻、企业、传承、税务和风险隔离等议题。
  • 风险测评与智库资源：辅助家庭识别财务结构、企业经营和传承安排中的风险点。

教育私享

不只面向子女教育，也可服务企业主自身学习、企业治理和接班规划。

  • 企业家学院：亚布力、企业家私塾、创新学堂、传承学院等学习资源。
  • 企业治理方案：围绕企业持续经营、家企关系、传承安排展开。
  • 子女教育：教育沙龙、名校研学、成长规划等资源，可按家庭情况选择性使用。
  • 对丁克家庭，这部分可重点理解为企业成长、认知升级和家庭长期规划资源。

艺术私享

偏圈层、审美和生活方式，用于高客家庭的文化体验和社交场景。

  • 艺术沙龙：参与艺术展览、艺术品之夜、文化品鉴等活动。
  • 艺术研学：通过艺术主题活动拓展生活方式和审美体验。
  • 艺术创作与拍卖交易：了解艺术品收藏、送拍、交易等基础路径。
  • 藏品保管：围绕收藏品管理和保管需求提供相关资源连接。

慈善私享

把家庭财富安排延展到公益、责任和价值传承。

  • 志愿者活动：参与泰康体系公益行动和志愿服务。
  • 幸福餐桌：围绕长者关怀、社区公益等主题的公益项目。
  • 慈善信托：为有长期公益规划的家庭提供结构化工具参考。
  • 溢彩公益专项基金：作为家庭慈善和社会责任表达的延展入口。

礼遇私享

覆盖出行、救援、活动和品质生活，在关键时刻提供服务接口。

  • 出行保障：境内道路救援、旅行相关咨询和出行场景服务。
  • 救援服务：部分场景可涉及全球医疗救援、紧急协助等资源连接。
  • 品质礼遇：酒店、活动、甄选品质游玩和高客生活方式资源。
  • 高客活动：泰康杯、生活家之夜等活动，增强家庭的圈层体验。

09 / 家庭能获得什么

这份方案把三类确定性放在一起

家庭真正需要看的，不只是单一收益率，而是这笔钱同时承担哪些长期功能：现金流、养老选择权、健康医疗资源入口。

现金流确定性

  • 陈露名下建立100万交3年的长期现金流。
  • 44岁起每年85,000元，60岁后进入养老保险金阶段。
  • 如果暂时不用，可进入万能账户继续沉淀。
  • 合同约定给付与万能账户演示收益分开理解，更便于家庭做长期安排。

资源确定性

  • 当前可对接500万家庭版一养老社区权益。
  • 核心为2个保证入住权与4个家庭优先入住权。
  • 医疗健康方面，可按资格连接私人医生、绿通、在线MDT、楚惠卡等服务。
  • 六大私享权益可作为健康、财富、生活方式和家庭规划的延展。

这份安排的价值，是把一笔长期资金同时变成三件事：每年可领取的现金流、未来养老社区的家庭选择权，以及平时和关键时刻可连接的医疗健康服务资源。
